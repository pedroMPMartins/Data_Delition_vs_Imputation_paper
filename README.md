# Removing Missing Data vs Imputation

This repository contains the LaTeX source code and the final PDF for my individual project in the **Introduction to Machine Learning** course.

## Overview

The paper provides a critical analysis of current missing data handling methodologies in machine learning, categorizing them into:
- Statistical Strategies
- Algorithmic Imputation
- Native Handling Strategies

Missing data represents a difficult challenge in real-world machine learning applications. When poorly handled, it can cause compromises in the resulting model's integrity and predictive performance. This paper explores the trade-offs between simple deletion strategies, traditional imputation, state-of-the-art deep learning approaches, and native algorithmic handling.

## Repository Structure

- `paper.tex`: The main LaTeX source file containing the paper content.
- `paper.pdf`: The compiled PDF of the paper.
- `notes.md`: Personal research notes and drafts used to write the paper.
- `Project1_Guidelines_group.pdf`: The official guidelines for the assignment.
- `New_IEEEtran_how-to.tex`: IEEE template reference file.

## How to Compile

To compile the LaTeX document into a PDF, you can use any standard LaTeX compiler like `pdflatex`:

```bash
pdflatex paper.tex
```

*(Note: You may need to run the compilation command a few times to properly generate references, tables, and citations.)*
